# Tube Backend

## HTTP Status Codes

### 2xx Success

- 200 OK

- 201 Created

### 3xx Redirection

- 301 Moved Permanently

- 302 Found (or 303 See Other)

### 4xx Client Errors

- 400 Bad Request

- 401 Unauthorized

- 403 Forbidden

- 404 Not Found

### 5xx Server Errors

- 500 Internal Server Error

- 502 Bad Gateway

- 503 Service Unavailable
